## 项目介绍

该项目是个人博客网站，使用 Hugo v0.150.0 构建，使用 blowfish 作为主题。

## Hugo 项目结构

由于部分文件夹没有内容，导致无法提交到 git 仓库

Hugo workdir
+ archetypes：定义内容页面的 front matter 模板
+ assets：存放需要被 Hugo 处理的静态文件
+ config：存放配置文件
+ content：存放页面内容
+ data：存放结构化数据
+ i18n：存放国际化相关文件
+ layouts：存放自定义的页面主题
+ static：存放不需要被 Hugo 处理的静态文件
+ themes：存放主题文件
